# User Management API

- Clone the repository.
- Use `npm install` to install the dependencies.

### Installation

```sh
$ git clone <repository_name>
$ cd <folder name>
$ npm install
```

Start the server using PM2 (in production) or nodemon (in development)

```sh
$ pm2 start bin/www
```

or

```sh
$ nodemon bin/www
```
